# Post-it Design

Plugin Jeedom permettant de créer des post-it affichables sur les Designs Jeedom.

Objectif :
- créer un équipement par post-it ;
- définir titre, message, couleur, largeur, hauteur et rotation ;
- visualiser un aperçu dynamique dans l’onglet Post-it ;
- redimensionner le post-it directement par les coins de l’aperçu ;
- choisir un Design cible depuis le post-it ;
- coller automatiquement le post-it sur le Design sans passer par l’édition manuelle du Design.

Version actuelle :
- page desktop de gestion ;
- rendu widget personnalisé via `toHtml()` ;
- aperçu dynamique redimensionnable ;
- création rapide d’un Design depuis le plugin ;
- collage automatique du post-it sur un Design Jeedom ;
- pas de dépendance ;
- pas de démon.
\n- rotation visuelle par curseur, boutons rapides et poignée au-dessus de l’aperçu.\n
