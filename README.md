# Post-it Design

Plugin Jeedom permettant de créer des post-it affichables sur les Designs Jeedom.

Objectif :
- créer un équipement par post-it ;
- définir titre, message, couleur, largeur, hauteur et rotation ;
- ajouter l’équipement sur un Design Jeedom pour l’afficher comme une note collée.

Premier état :
- plugin installable ;
- page desktop de gestion ;
- rendu widget personnalisé via `toHtml()` ;
- pas de dépendance ;
- pas de démon.
